The **Repeater Tab** allows you to manually modify and replay individual requests, making it essential for probing specific endpoints and verifying vulnerabilities.

## How to Use

1. **Send to Repeater**: Right-click a request in the Traffic tab and select "Send to Repeater".
2. **Modify**: Edit the method, URL, headers, or body in the request pane.
3. **Send**: Click "Send" to execute the request.
4. **Analyze**: View the response in the response pane.

## Features

-   **History**: Navigate through previous versions of your modified requests.
-   **Tabs**: Open multiple repeater tabs for different requests.
-   **Pretty Print**: Automatically format JSON, XML, and HTML responses.
-   **Render**: View the HTML response as it would appear in a browser.

## Use Cases

-   **API Testing**: Manually test API endpoints with different parameters.
-   **Vulnerability Verification**: Confirm a suspected vulnerability found by Intruder.
-   **Logic Bugs**: Test for business logic flaws by manipulating request data.
